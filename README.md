# yii-template

> 基于nodejs开发的前端项目模板，依赖nodejs开发环境，为自定义组件、技术研究、二次开发以及代码讲解开发的项目母版，其他各项目都对此进行fork

## 编译过程

``` bash
# 安装依赖
npm install

# 启动热加载服务，访问地址http://localhost:8080
npm run dev

# 在生产模式下构建项目，内容包括代码校验、压缩以及混淆，
npm run build

# 运行单元测试
npm run unit

# 运行e2e测试
npm run e2e

# 运行所有的测试
npm test
```
